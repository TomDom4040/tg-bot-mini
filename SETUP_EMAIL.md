# Настройка Email для сброса пароля

## Шаг 1: Настройка Gmail

1. **Включите двухфакторную аутентификацию** в вашем Google аккаунте
2. **Создайте пароль приложения:**
   - Перейдите в [настройки Google аккаунта](https://myaccount.google.com/)
   - Безопасность → Пароли приложений
   - Выберите "Почта" и "Другое устройство"
   - Введите название: "Admin Panel"
   - Скопируйте сгенерированный пароль (16 символов)

## Шаг 2: Настройка переменных окружения

Отредактируйте файл `.env` в папке проекта:

```bash
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=ваш-email@gmail.com
EMAIL_PASS=ваш-пароль-приложения
EMAIL_FROM=ваш-email@gmail.com
BASE_URL=http://165.232.139.129:3000
```

**Замените:**
- `ваш-email@gmail.com` на ваш реальный Gmail
- `ваш-пароль-приложения` на пароль приложения из шага 1

## Шаг 3: Перезапуск сервера

```bash
pkill -f "node index.js"
cd /root/Tg_BOTs/tg-bot-mini
node index.js
```

## Тестирование

1. Откройте: http://165.232.139.129:3000/login
2. Нажмите "Забыли пароль?"
3. Введите email, который настроен в EMAIL_USER
4. Проверьте почту - должно прийти письмо со ссылкой для сброса
